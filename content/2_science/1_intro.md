Jupyter Book, utilizing the MyST engine, is specifically designed to create interactive web-based scientific publications. But, without additional effort, it allows to create a PDF version that can be submitted to a journal. Several [templates](https://github.com/orgs/myst-templates/repositories) have been created already.

To see the available options for a specific template, run `myst templates list <template_name> --<type>` where `<type>` can be either `tex`, `typst` or `docx`.
